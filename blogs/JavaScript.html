    • Introducción a JavaScript

    JavaScript, a menudo abreviado JS, es un lenguaje de programación que forma parte de las tecnologías básicas de la World Wide Web, junto con HTML y CSS. Nos permite añadir interactividad a las páginas. Por ejemplo, es posible que haya visto deslizadores, alertas, interacciones de clic, ventanas emergentes, etc. en distintos sitios web; todo ello se construye con JavaScript. Además de ser utilizado en el navegador, también se utiliza en otros entornos que no son navegadores, como Node.js para escribir código del lado del servidor en JavaScript, Electron para escribir aplicaciones de escritorio, React Native para aplicaciones móviles, etc.



    Historia de JavaScript

    JavaScript fue creado inicialmente por Brendan Eich de NetScape y fue anunciado por primera vez en un comunicado de prensa por Netscape en 1995. Tiene una extraña historia de nombres; inicialmente, fue llamado Mocha por el creador, que más tarde fue renombrado LiveScript. En 1996, aproximadamente un año después del lanzamiento, NetScape decidió renombrarlo a JavaScript con la esperanza de sacar provecho de la comunidad Java (aunque JavaScript no tenía ninguna relación con Java) y lanzó Netscape 2.0 con el soporte oficial de JavaScript.

    • Versiones de Javascript

    JavaScript fue inventado por Brendan Eich, y en 1997 se convirtió en un estándar ECMA. ECMAScript es el nombre oficial del lenguaje. Las versiones de ECMAScript son ES1, ES2, ES3, ES5 y ES6.

    • JavaScript Variables

    La mayor parte del tiempo, una aplicación JavaScript necesita trabajar con información. Para almacenar y representar esta información en la base de código JavaScript, utilizamos variables. Una variable es un contenedor para un valor.

  • ¿Cómo declarar variables de diferentes maneras en JavaScript?

    • Palabra clave [ let ]

    La declaración let declara una variable local de bloque, opcionalmente inicializándola a un valor.

    Palabra clave [ const ]

    Las constantes están limitadas por bloques, al igual que las variables declaradas con la palabra clave let. El valor de una constante no puede modificarse mediante reasignación (es decir, utilizando el operador de asignación), y no puede volver a declararse (es decir, mediante una declaración de variable). Sin embargo, si una constante es un objeto o una matriz, sus propiedades o elementos pueden actualizarse o eliminarse.
